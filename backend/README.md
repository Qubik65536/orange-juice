# Backend

This is the backend of the OJ system, built with [Rust](https://www.rust-lang.org) and [Axum](https://docs.rs/axum).

## Setup

Make sure to install the dependencies:

```bash
cargo build
```

## Running the Server

Start the server on `http://localhost:8000`:

```bash
cargo run
```

## Testing

This project uses [Rust's built-in testing framework](https://doc.rust-lang.org/book/ch11-00-testing.html) for unit and integration tests.

Run all tests:

```bash
cargo test
```
